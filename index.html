FAQ

Quick answers about the interactive navigation and responsiveness.

How does the nav change on scroll?

JavaScript listens for scroll events and adds/removes the .scrolled class when you pass 50px.

Does the mobile menu work?

Yes. The hamburger button toggles an accessible mobile panel and closes it when you click a link.

Are hover effects smooth?

Yes. Menu item underline and color changes use CSS transitions for smooth visual feedback.

Is it responsive?

Uses media queries and grid/flex to adjust at tablet and mobile sizes.
