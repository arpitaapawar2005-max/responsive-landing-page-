# Responsive Landing Page — Task (Prodigy Infotech)

A fully responsive landing page built with **HTML, CSS, and JavaScript**, featuring an interactive, fixed navigation bar.

## Live Demo
🔗 https://arpitaapawar2005-max.github.io/responsive-landing-page-/

## Task Requirements Covered
- **Fixed navigation bar** — stays pinned to the top and visible on the whole page using `position: fixed`.
- **Scroll-based style change** — a JavaScript scroll listener toggles a `.scrolled` class once the user scrolls past 50px, switching the nav from transparent to a solid, blurred background with a shadow.
- **Hover-based style change** — CSS `:hover` transitions animate an underline and color change on each nav link.
- **Responsive layout** — CSS Grid/Flexbox with media queries adapts the layout for desktop, tablet, and mobile; the nav collapses into a hamburger menu below 760px.
